# PANE // LANDING

> **Mixing Style With Focus.**

This is the web portal for **Pane**, the Capacitor-based Android productivity tool. It serves one purpose: to look as sharp as the app itself while delivering the latest APK to users without me having to lift a finger after a release.

It is built with **React**, styled with **Tailwind CSS**, and engineered to feel "heavy" and premium.
